Deklarasi Angka Desimal

Angka Desimal

var angkapertama = 1234567890 ;

var angkakedua = 42 ;

var angkaketiga = 088 ; // ini adalah angka desimal

var angkakeempat = 0777 ; // ini adalah angka oktal

Untuk mendeklarasikan angka desimal (basis 10), langsung ditulis angkanya saja. Jika sebuah angka desimal dimulai angka 0 dan diikut angka 1-7 maka angka tersebut dikenali sebagai bilangan oktal (basis 8), Sedangkan angka 0 yang diikuti angka 8 keatas dianggap angka desimal
